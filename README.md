# DiscordAppInjector
With the help of this API you can run files while Discord is opening.

---------------------------------

### Inject ShellCommand

```java
Injector.getINSTANCE().injectShellCommand("start calc.exe");
```

<p></p>

---------------------------------

###  Inject Asar File

```java
Injector.getINSTANCE().injectAsar("Path/To/file.asar");
```
